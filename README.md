**CompTIA Network+ Study Hub — N10-009**

A self-contained, offline-first study hub for the CompTIA Network+ N10-009 exam. No server, no login, no cloud dependency — just open the HTML file in any browser and study.

No installation. No internet required after the initial font load.

**Cross-Device Workflow**

Because this is a static HTML file, your progress lives in the browser it was opened in. To carry it to another device:

Click ⬇ Export in the top-right header — saves a .json file
On the new device, open the hub and click ⬆ Import — select the JSON file
All scores, checkmarks, quiz history, and flashcard ratings transfer instantly

**Settings & Reset**

On the Dashboard tab, a Settings & Data card lets you:

Set Exam Date — updates the countdown timer and study pace calculator
Reset All Progress — wipes all localStorage data (export first!)
